Live demo — pipeline inspector

A genuinely interactive demo — no server. Edit the JSON Payload and watch the whole pipeline recompute in your browser, stage by stage: the authored Template is bound against the Resolved payload (the payload enriched by the Resolve phase with $derived.* values), assembled into a renderer-agnostic DocumentNode tree (resolving each Clause reference to a concrete version), and finally rendered to HTML — the same tree-assembly and render path the library uses server-side (binding, clause resolution, key-value rows, signatures), minus the audit Snapshot (which needs a Node-only hash). The Template and its two Clauses are seeded in-memory for this page — see the Authoring guide for how a real catalog on disk looks.
